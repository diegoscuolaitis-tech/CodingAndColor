# CodingAndColor
Autore: Diego Simonetti e Monica Ciuchetti.

Progetto: Sito web che permette di inserire e leggere 3 numeri, compresi tra 0 e 255, e scrivere un messaggio a piacere che una
volta inseriti i 3 numeri verrà modificato il colore del testo in base alla codifica RGB dei 3 numeri scelti. 
In caso di scelta dei numeri superiore a 255 o minore di 0 il sito darà un errore per via del numero dato non interno al range 
della codifica RGB.
